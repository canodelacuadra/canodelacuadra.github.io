# Laravel 0
Laravel es un framework web basado en PHP. Se denomina framework para artesanos web. La base del framework propone los fundamentos, liberando al artesano web para crear sin preocuparse por las pequeñas cosas.

Laravel recurre a  un modelo de arquitectura MVC (Model View Controller) y gira en torno a las operaciones CRUD (Create, Retrieve, Update, Delete).

Laravel 8, se lanzó el 8 de septiembre de 2020.
